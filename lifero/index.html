D-CLIC, formez-vous au numérique avec l'OIF

D-CLIC est une plateforme d’apprentissage à distance développée par l’Organisation Internationale de la Francophonie (OIF). Elle propose des parcours de formation au numérique en français, accessibles à tous et adaptés aux besoins du marché actuel.

Les parcours disponibles sur D-CLIC sont :

  • Métiers du numérique
    • Session de Mai 2025
    • Session de Juillet 2025
      1. Développement Web
      2. Développement Mobile
      3. Marketing Numérique
  • Compétences numériques de base
  • Gouvernance du numérique
  • Opérations de formation
  • Formation des formateurs

La plateforme regroupe des ressources éducatives libres, des modules interactifs, des exercices pratiques et des mini-projets pour permettre à chacun de progresser à son rythme.

Mon expérience :
J’ai eu la chance de participer à la session de juillet 2025, axée sur le développement web. Chaque semaine, nous avons abordé des thématiques variées : structuration de pages HTML, mise en forme avec CSS, interactivité avec JavaScript, bonnes pratiques pour l’accessibilité et la compatibilité mobile.

Cette formation en ligne, organisée par l’OIF, m’a permis d’acquérir des compétences solides et de réaliser plusieurs projets concrets. Les différentes semaines sont détaillées ci-dessous pour illustrer les acquis et les leçons apprises.

Semaine 1 : Introduction au web

Cette première semaine sur D-CLIC a permis de découvrir les fondamentaux du numérique et de la programmation. Nous avons commencé par la réalisation d’une carte mentale pour organiser nos idées et visualiser les concepts clés du développement web.

Les points abordés :

  • Présentation de la plateforme D-CLIC et de ses parcours ;
  • Introduction aux métiers du numérique et aux compétences de base
  • Utilisation d’outils collaboratifs et de ressources éducatives libres ;
  • Création d’une carte mentale pour structurer les connaissances acquises ;
  • Premiers exercices pratiques pour manipuler des outils numériques.

Cette étape a été essentielle pour poser les bases et faciliter la compréhension des modules suivants. Pour voir la carte mentale réalisée :cliquez ici

Semaine 2 : Introduction HTML 5

Cette semaine sur la plateforme D-CLIC a été dédiée à l’apprentissage des bases du développement web. Nous avons suivi des modules interactifs sur HTML pour structurer les pages, CSS pour la mise en forme, et JavaScript pour rendre les pages dynamiques.

Les activités proposées incluaient :

  • Création de pages web simples et structurées ;
  • Utilisation des balises sémantiques et des formulaires ;
  • Personnalisation du style avec des feuilles CSS ;
  • Ajout d’interactivité avec des scripts JavaScript (ex : boutons, alertes, calculs) ;
  • Découverte des bonnes pratiques pour l’accessibilité et la compatibilité mobile.

Grâce à ces exercices, j’ai pu réaliser des mini-projets et mieux comprendre le fonctionnement d’un site web moderne. Pour voir un exemple de synthèse réalisée cette semaine :cliquez ici

Semaine 3 : Introduction aux feuilles de style CSS

Durant cette troisième semaine sur D-CLIC, nous avons approfondi nos connaissances en feuille de styles CSS et exploré des concepts avancés.

Les points clés abordés :

  • Comprendre le concept de mise en forme en utilisant les feuilles de style ;
  • Maîtriser la mise en forme en utilisant CSS pour créer des pages web visuellement attrayantes et esthétiques ;
  • Reconnaître et mettre en œuvre les divers types de sélecteurs CSS, tels que les sélecteurs de classe, les sélecteurs d'ID et les sélecteurs d'éléments, afin de cibler spécifiquement les éléments à styliser ;
  • Introduction aux tableaux et aux objets pour gérer des collections de données ;
  • Appliquer les sélecteurs CSS aux différentes balises HTML.

Ces compétences m’ont permis de créer des pages web plus interactives et fonctionnelles. Pour voir un exemple d’exercice réalisé cette semaine : cliquez ici

Semaine 4 : Introduction aux JavaScript

Au cours de cette quatrième semaine sur D-CLIC, nous avons exploré les bases en JavaScript et son intégration dans une page web. Nous avons appris à utiliser des Opérations avec JavaScript pour effectuer des calculs mathématiques et des opérations sur les chaînes de caractères.

Les points clés abordés :

  • Intégration d'un script Javascript dans une page HTML ;
  • Création des variables et comprendre leur portée dans JavaScript ;
  • Utiliser des opérateurs JavaScript pour effectuer des calculs mathématiques et des opérations sur les chaînes de caractères ;
  • Créaction des structures conditionnelles, des boucles et manipuler des tableaux ;
  • Utilisation des fonctions JavaScript pour organiser votre code de manière modulaire et réutilisable.

Ces compétences m’ont permis de créer des page web plus robustes. Pour voir un exemple d’exercice réalisé cette semaine : cliquez ici

Semaine 5 : les objets Javascript

Au cours de cette cinquième semaine sur D-CLIC, nous avons abordé les objets Javascript en utilisant des éléments adéquats. Nous avons découvert comment ces objets peuvent simplifier le processus d'effectuer des opérations complexes.

Les points clés abordés :

  • Comment intégrer des formulaires dans des pages HTML en utilisant des éléments adéquats ;
  • Comment ajouter des éléments de formulaire tels que des champs d'entrée, des boutons de soumission, etc. en fonction des types de données à saisi ;
  • Gestion des objets standards de JavaScript tels que Math, String, Date, et Array, etc ;
  • Comment accéder aux propriétés et aux méthodes de ces objets ;
  • Comment utiliser ces objets pour effectuer des opérations courantes telles que le calcul mathématique, la manipulation de chaînes de caractères, la gestion des dates, et la manipulation de tableaux. ;
  • Réalisation d’exercices pratiques pour consolider les acquis

Ces compétences m’ont permis de créer des pages web plus dynamiques et réactives. Pour voir un exemple d’exercice parmi tant d'autres réalisé cette semaine : cliquez ici

Semaine 6

Lors de cette sixième semaine sur D-CLIC, nous avons exploré les bonnes pratiques de développement et les outils de collaboration. Nous avons appris à utiliser des systèmes de contrôle de version et à travailler en équipe sur des projets personnel

Les points clés abordés :

  • Introduction aux systèmes de contrôle de version (ex : Git) ;
  • Utilisation de plateformes collaboratives (ex : GitHub, GitLab) ;
  • Gestion des branches et résolution des conflits ;
  • Revue de code et bonnes pratiques de développement ;
  • Réalisation d’exercices pratiques pour consolider les acquis ;
  • Réalisation du projet final.

Objectif

Mon objectif en suivant cette formation sur la plateforme D-CLIC était d'acquérir des compétences solides en développement web afin de pouvoir créer des sites et applications modernes, interactifs et accessibles. Grâce aux différents modules et exercices pratiques, j'ai pu progresser rapidement et réaliser plusieurs projets concrets.

Je souhaite désormais continuer à approfondir mes connaissances, explorer de nouvelles technologies et contribuer à des projets open source. Mon ambition est de devenir un développeur web compétent et polyvalent, capable de répondre aux besoins du marché actuel et de participer activement à la communauté du développement.

Je suis convaincu que cette formation constitue une base solide pour atteindre mes objectifs professionnels et personnels dans le domaine du numérique.
